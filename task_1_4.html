С атрибутом async парсинг HTML продолжается до тех пор, пока браузер не загрузит файл сценария по сети, поэтому парсинг и загрузка скрипта происходят параллельно. Как только скрипт становится доступным в браузере, парсинг HTML приостанавливается и скрипт выполняется. C атрибутом defer, упомянутым в теге скрипта, файлы скрипта загружаются параллельно, пока продолжается парсинг HTML. Но выполнение откладывается до завершения парсинга HTML.
